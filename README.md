# 🇺🇸 H1B Visa Analytics Dashboard & Data Analysis

## 📌 Overview

This project analyzes **H1B Visa application data** to uncover trends in job roles, wages, employer demand, and case status across the United States. It combines **Python-based data analysis** with an **interactive Power BI dashboard** for better insights.

---

## 🎯 Objectives

* Analyze visa application trends
* Identify top job roles and employers
* Study wage distribution
* Explore state-wise job demand
* Visualize insights using dashboards

---

## 🛠️ Tech Stack

* Python (Pandas, NumPy)
* Matplotlib & Seaborn
* Jupyter Notebook
* Power BI

---

---

## 📊 Key Insights

* Majority of applications are **Certified**
* **Software Developers** are the most in-demand role
* **California and Texas** lead in job opportunities
* Average wage is around **$123K**
